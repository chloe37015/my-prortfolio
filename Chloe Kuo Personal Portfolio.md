# Chloe Kuo Personal Portfolio

這是一個為 **郭姿妤 (Chloe)** 量身打造的個人作品集網站。採用現代化的藍白色調漸層設計，並具備響應式佈局，完美展示學術研究與商業分析成果。

## 🚀 快速開始

### 本地預覽
直接使用任何瀏覽器開啟 `portfolio.html` 即可查看。

### GitHub Pages 部署教學
1. 在 GitHub 上建立一個新的儲存庫 (Repository)，名稱可以叫 `my-portfolio`。
2. 將 `portfolio.html` 重新命名為 `index.html`。
3. 將所有文件 (`index.html`, `README.md`, `.gitignore`, `LICENSE`) 上傳至該儲存庫。
4. 前往儲存庫的 **Settings > Pages**。
5. 在 **Build and deployment > Branch** 選擇 `main` 分支並點擊 **Save**。
6. 等待幾分鐘後，您的作品集將在 `https://<您的帳號>.github.io/my-portfolio/` 上線！

## ✨ 功能亮點
- **漸層視覺設計**：專業的藍白色系漸層背景，提升視覺層次感。
- **互動式作品集**：點擊卡片可開啟詳細的 Modal 視窗，查看專案深度說明。
- **PDF 報告連結**：整合了雲端 PDF 連結，可直接在線上閱讀完整報告。
- **LinkedIn 複製功能**：避開 LinkedIn 連結封鎖，提供「點擊複製」的貼心互動。
- **響應式佈局**：在手機、平板與電腦上均有良好的閱讀體驗。

## 📂 檔案結構
- `portfolio.html`: 核心網站文件（包含 HTML/CSS/JS）。
- `README.md`: 專案說明文件。
- `.gitignore`: Git 忽略清單。
- `LICENSE`: MIT 開源授權協議。
